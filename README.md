This is a clean, modern Pokédex Showcase landing page. It features a split-screen design, smooth scroll snapping, and a responsive layout that highlights six iconic Generation I Pokémon.

🚀 Features
Split-Screen Design: A dynamic layout with a 60/40 color split tailored to each Pokémon's theme.

Smooth Scroll Snapping: The page locks onto each Pokémon section as you scroll for a premium "app-like" feel.

Custom Typography: Leverages the "Montserrat" Google Font for a bold, high-end editorial look.

Responsive Header: A fixed navigation bar containing the official Pokémon logo and a sleek layout.

Interactive Visuals: Pokémon images feature a floating drop-shadow effect and a scale-up hover animation.

No-JS Interaction: All visual transitions and layouts are handled purely via CSS.

📂 File Structure
index.html: Contains the structural layout, including the fixed header and six distinct sections for Pikachu, Charmander, Squirtle, Bulbasaur, Eevee, and Gengar.

style.css: Contains all visual styling, including CSS variables for color themes, Flexbox/Absolute positioning, and @media queries for mobile responsiveness.

Assets:

one.webp: The primary hero image for the Pikachu section.

pikachu.png, charmander.png, etc.: Transparent PNG assets for the character showcase.
